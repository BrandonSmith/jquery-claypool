Claypool Documentation Resources
API Documentation

The API Documentation is our highest priority now and it is constantly being updated with more information to help you understand how to best make use of Claypool.

The Api Documentation is itself a small MVC application running in your browser so you can are examine the sources in firebug to get an idea of how a simple application can benifit from just a little bit of help from Claypool.
